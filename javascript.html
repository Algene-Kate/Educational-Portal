Example of JavaScript Code

// Simple JavaScript program
console.log("Hello, World!");

function greet(name) {
    return `Hello, ${name}!`;
}

// Using template literals
console.log(greet("Julianna"));

// Arrow function example
const multiply = (a, b) => a * b;
console.log(`5 x 3 = ${multiply(5, 3)}`);

// Working with arrays
const fruits = ["Apple", "Banana", "Orange"];
fruits.forEach(fruit => console.log(`I like ${fruit}s`));

// DOM manipulation example
document.addEventListener('DOMContentLoaded', () => {
    const button = document.querySelector('#myButton');
    button.addEventListener('click', () => {
        alert('Button clicked!');
    });
});

Why Learn JavaScript?

JavaScript is a powerful language with wide-ranging applications. From web development to mobile apps and server-side programming, JavaScript has established itself as one of the most important programming languages today. With its ease of use, lightweight nature, and cross-platform capabilities, JavaScript continues to play a significant role in shaping modern web applications.

Learning JavaScript opens doors to countless opportunities in the tech industry. As the language of the web, it remains in high demand among employers, and the vibrant ecosystem of frameworks and libraries ensures that JavaScript developers are equipped with the tools they need to build anything from simple websites to complex, enterprise-grade applications.
